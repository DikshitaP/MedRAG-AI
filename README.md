# MedRAG-AI
MedRAG AI is an intelligent medical chatbot built using LLMs, LangChain, and Pinecone with a RAG-based architecture. It delivers context-aware medical responses through semantic search and vector retrieval. The project is deployed on AWS to ensure scalability and real-time interaction.

# How to run?

### steps

```bash
git clone https://github.com/DikshitaP/MedRAG-AI.git
```
### step 01: Create  a code environment after cloning the repository

```bash
conda create -n MediRAG-AI python=3.10 -y
```

```bash
conda activate MediRAG-AI
```
### step 02: install the requirements

```bash
pip install -r requirements.txt
```
